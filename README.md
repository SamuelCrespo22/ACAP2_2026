# We Need More Butterflies! - ACA Project 2

## Project Overview

This project is the second practical assignment for the Advanced Machine Learning course (MIACD, University of Coimbra).

The primary goal is to improve the performance of a pre-determined image classifier (baseline CNN) by augmenting the dataset with images generated by generative models. The dataset consists of 75 distinct classes of butterflies, comprising over 5199 labeled images.

**Key Objectives:**

- Prepare a machine learning pipeline for image classification.
- Analyze the provided baseline CNN model.
- Explore and implement generative machine learning models to perform data augmentation:
  - AutoEncoders
  - Generative Adversarial Networks (GANs)
  - Diffusion Models (optional)
- Train the baseline model with the original dataset.
- Train the baseline model with augmented datasets (original + generated samples).
- Compare the results and evaluate the improvement without tweaking or changing the baseline classifier architecture, optimizer, or loss function.
- Predict on the test set and submit the results to the Kaggle competition.

## How to Run

1. **Exploratory Data Analysis & Baseline Check**: Open `notebooks/TP2-students.ipynb` (or just `TP2-students.ipynb`) to explore the dataset and the initial baseline code.
2. **Train Generative Models**: Implement and run scripts in `src/models/` to train GANs, AutoEncoders, or Diffusion models on the `data/train/` dataset.
3. **Generate Augmented Data**: Use your trained generative models to create new synthetic images and save them in `generated_data/`.
4. **Train Augmented Classifier**: Train the baseline classifier using the combined dataset (`data/train/` + `generated_data/`). Ensure the baseline architecture remains unchanged.
5. **Evaluate & Predict**: Evaluate the models on the validation split and use the best model to generate predictions on `data/test/`. Format the output according to the competition's requirements and save it to `submissions/`.
